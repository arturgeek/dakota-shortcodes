Common Features Asked by different clients to show item cart count, easy to implement in different builders

== Description ==

The main puporse of this set of shortcodes is to facilitate a way to implement common requirements from different clients 
when we are building their eCommerce sites, this in order to improve the time implementing the solutions as well as adding value 
with an Out of The Box enhancer to woocommerce.
